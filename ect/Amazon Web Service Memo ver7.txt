EC2 [ Elastic Compute Cloud ]
Amazon Machine Image ( AMI )는 운영체제 위에 애플리케이션이 있는 형태로
바이너리 코드로 구성하여 여러 개의 가상 머신에 배포할 수 있다.
	예를 들어 딥러닝 애플리케이션을 만들었다면 window 운영체제 위에 있는 딥러닝 애플리케이션을
	바이너리 코드로 바꾸어서 가상 머신에 올리면 그것들을 AWS에서 사용한다.

Auto Scaling [ 자동 확장 매커니즘 ]
	두 개의 서버가 필요하다고 할 때
	한 개의 서버를 최소 크기,
	두 개의 서버를 목표 용량
	예비로 준비된 2 + 2를 최대 크기라고 한다.
	이렇게 필요할 때 마다 자동으로 서버 크기를 확장하는 매커니즘을
	Auto Scaling이라고 한다.
